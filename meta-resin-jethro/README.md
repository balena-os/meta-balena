# Resin.io layer for fido supported boards

## Description
This repository enables building resin.io for fido supported machines.

## Supported machines
raspberrypi
raspberrypi2
parallella-hdmi-resin
nitrogen6x
odroid-c1
odroid-xu3
intel-corei7-64
cubox-i
beaglebone
vab820-quad
zc702-zynq7
